# vscode-bigrapher

## About

This is VSCode extension for [BigraphER](https://uog-bigraph.bitbucket.io/)

## Features

- Syntax highlighting
- Comment out / uncomment

## Install 

- e.g. `\Users\<USERNAME>\.vscode\extensions\seelx3.bigrapher-0.0.1` (Windows)

## Info

- This extension is in development.
